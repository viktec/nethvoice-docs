# NethVoice SaaS Services

Nethesis offers two cloud-based deployment options for NethVoice: **NethServer 8 SaaS** for infrastructure management and **NethVoice as a Service** for a fully managed application service.

All SaaS services are available inside [NethShop](https://nethshop.nethesis.it).

:::warning
SaaS services are available only for Nethesis resellers.
If you are not a reseller and you want to become one, please contact [sales](mailto:sales@nethesis.it)
:::

## NethServer 8 SaaS {#nethserver-8-saas}

**NethServer 8 SaaS** is a managed cloud infrastructure service provided by Nethesis to partners. It includes a ready-to-use NethServer 8 node available in multiple capacity tiers, allowing you to deploy and manage any application from the certified Software Center (NethVoice, Firewall Controller, Nextcloud, Mattermost, WebTop, etc.).

### Service Model

- **Managed Infrastructure**: Nethesis manages the NethServer node, OS updates, monitoring, and backups
- **Multi-Application**: Install any certified Software Center application on the same node
- **Multi-Tenant**: You can deploy multiple NethVoice instances on a single node
- **Clustering**: NethServer 8 SaaS nodes can only connect to other Nethesis SaaS nodes

### Getting Started

1. Visit [NethShop](https://nethshop.nethesis.it) to select a capacity tier
2. Configure your preferred tier
3. Once provisioned, install and configure your applications via the Software Center

## NethVoice as a Service {#nethvoice-as-a-service}

**NethVoice as a Service** is a fully managed cloud service where Nethesis handles all infrastructure, system management, monitoring, backups, and updates. Partners manage only the application-level configuration and user administration.

:::info
NethVoice as a Service provides a turnkey solution: pay only for the number of users you need, and Nethesis handles everything else.
:::

### Service Features

- **Fully Managed**: Nethesis manages infrastructure, OS, updates, monitoring, backups, and support
- **Application Management**: You configure and administer NethVoice, create users, and manage profiles
- **Automatic Backups**: 28-day retention on encrypted, redundant cloud systems
- **Level 2 Support**: Nethesis provides second-level technical support
- **Multi-Tenant**: Deploy multiple NethVoice instances across your customers

### Service Administration

#### Responsibilities: What Nethesis Provides

- **Backup Management**: Automatic 28-day retention on encrypted, redundant cloud systems
- **Monitoring**: 24/7 system monitoring and alerts
- **Updates**: Automatic security and feature updates
- **Level 2 Support**: Second-level technical assistance for unresolved issues

#### Responsibilities: What the Partner Provides

- **DNS Configuration** (if using custom domain): Configure DNS records at your provider to point to the provided IP addresses
- **Migration Support** (optional): Partners can request Nethesis to handle migration from previous NethServer versions (paid service, not included in standard NethVoice as a Service subscription)
- **Application Configuration**: Set up users, profiles, trunks, routes, and other NethVoice-specific settings
- **Level 1 Support**: Provide first-level support to your end users using admin logs and debug tools
- **Issue Escalation**: Report issues to Nethesis with detailed logs and reproduction steps when Level 1 troubleshooting is exhausted

### Backup Configuration

Backups are **automatic and transparent**:

- **Frequency**: Continuous
- **Retention**: 28 days
- **Location**: Encrypted, redundant cloud systems
- **Recovery**: Contact Nethesis support for recovery requests

## Choosing Between the Two Services

| Feature | NethServer 8 SaaS | NethVoice as a Service |
|---------|---|---|
| **Infrastructure** | Managed by Nethesis | Managed by Nethesis |
| **Applications** | Any Software Center app | NethVoice only |
| **Management** | Full control via Web UI | Application-level only |
| **Multi-instance** | Multiple apps on one node | Multiple NethVoice instances |
| **Use case** | Infrastructure control, multiple services | NethVoice-focused, turnkey solution |
| **Billing** | Annual | Quarterly |
| **Best for** | Partners, resellers | Service providers, end customers |
