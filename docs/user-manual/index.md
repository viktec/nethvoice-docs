Welcome to the user manual for NethVoice. This section contains user-facing guides for the NethVoice web client and desktop client. Use the links below to jump to the relevant product documentation and quickstart pages.

- [NethVoice CTI](/docs/user-manual/nethcti/) — web client for browser-based call management and collaboration.
- [NethLink](/docs/user-manual/nethlink/) — desktop application (Windows/macOS/Linux) for native call control.
- [NethVoice Mobile App](/docs/user-manual/mobile_app/) — mobile application for iOS and Android devices to make and receive calls on the go.
